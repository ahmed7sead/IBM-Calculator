### Synchronized Input Controls
Both number inputs and range sliders are perfectly synchronized, allowing users to input data in their preferred way.

### History Management
- Stores calculation history in localStorage
- Displays date, time, gender, height, weight, BMI, and status
- Individual delete or clear all options
- Automatically limits to 20 most recent entries

### Theme Persistence
User's theme preference (dark/light) is saved and automatically applied on page reload.

### Responsive Design
Gracefully adapts to different screen sizes:
- **Desktop**: Three-column layout with side-by-side gender inputs
- **Tablet/Mobile**: Stacked vertical layout for optimal mobile experience
